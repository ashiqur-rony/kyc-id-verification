# Image Manipulation Detection Models
Models trained on Kaggle. The source notebook is available at https://www.kaggle.com/code/ghumkumar/image-manipulation-detection-model.

### model_forgery.pth
Forgery detection model trained on the CASIA v2.0 dataset.  
Source: https://www.kaggle.com/datasets/ghumkumar/model-trained-with-casia-2-0-images
### model_scripted.pth
Scripted version of the model for deployment.  
Source: https://www.kaggle.com/datasets/ghumkumar/model-trained-with-casia-2-0-images
